Coursera app clone

I developed a clone of the Coursera for Campus web app to demonstrate my front-end development skills. This project focused on accurately replicating the design of the original platform, creating a responsive and visually appealing interface. I used modern technologies such as HTML5, CSS3, and JavaScript to build a dynamic user experience that closely mirrors the look and feel of the actual Coursera app. The app features a comprehensive course catalog, complete with search and filter options, ensuring a smooth and intuitive navigation experience. By focusing on the visual elements and layout, I was able to create an interface that is both engaging and user-friendly. This project highlights my ability to design and implement high-quality, responsive web interfaces with a strong emphasis on user experience and aesthetic fidelity, showcasing my proficiency in front-end development.
