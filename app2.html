Sky-Scanner App

Sky Scanner is a cutting-edge mobile application designed to revolutionize cloud classification using deep learning and generative AI techniques. This app features a user-friendly interface built with Flutter and Dart, ensuring a seamless and efficient user experience. By integrating advanced deep learning techniques such as the Inception v3 architecture, Sky Scanner achieves an impressive 96% accuracy in cloud image classification, providing users with precise and reliable results. To address the challenge of limited datasets, the app employs StyleGAN Generative AI techniques to generate synthetic cloud images, enriching the dataset and enhancing model performance. The app offers dynamic visualizations and analyses, enabling users to explore cloud patterns, trends, and classifications in real time.
This predictive capability empowers researchers, meteorologists, and enthusiasts to gain deeper insights into atmospheric conditions. By understanding cloud types and their implications, users can make informed decisions in fields such as weather forecasting, environmental monitoring, and education. The Sky Scanner app showcases my expertise in mobile app development, deep learning, and generative AI, providing a practical and impactful solution for real-world challenges.
