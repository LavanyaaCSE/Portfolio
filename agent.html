Selections AI Agent

I developed an advanced AI-powered web application leveraging OpenAI’s Gemini model to analyze and evaluate students’ GitHub and LeetCode profiles, highlighting my expertise in artificial intelligence, machine learning, and web development. This application is designed to classify students into two categories: "Ready for Placement" and "Not Ready for Placement." By analyzing coding activity, project quality, and problem-solving performance, the app provides tailored insights to enhance students’ readiness for technical placements.
For students categorized as "Ready for Placement," the application delivers comprehensive performance analysis and generates test questions to refine their skills further, helping them prepare for advanced challenges. For those in the "Not Ready for Placement" category, the app provides a detailed improvement roadmap, actionable insights, and resources to address specific weaknesses.
The application features an intuitive and visually appealing dashboard that presents analytics and recommendations in a user-friendly manner, making it easy for students to understand and act on the feedback. This project demonstrates my ability to design and implement intelligent systems that combine cutting-edge AI models with practical applications, emphasizing user experience, precision, and a structured approach to career readiness. It serves as a testament to my proficiency in leveraging AI to solve real-world problems and creating impactful solutions for users.
